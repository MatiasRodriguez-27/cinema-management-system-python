# 🎬 Cinema Management System (Python)

Console-based application developed in Python for the management of a cinema, including user administration, movie listings, and seat reservation system.

## 🚀 Features

* User authentication system with roles (Admin, User, Guest)
* Movie management (CRUD operations: create, update, delete)
* Seat reservation system with real-time availability
* Matrix-style visualization of seats
* Data persistence using text files
* Input validation and error handling
* Interactive console menu

## 🛠️ Technologies

* Python
* File handling
* Data structures (lists and dictionaries)

## 🎯 Purpose

This project was developed to practice programming logic, data structures, and system design, simulating a real-world management system.

## ▶️ How to Run

1. Clone the repository
2. Open the project in Python
3. Run the main file:
   ```bash
   python main.py
